CS Projects

Creating a Virtual Memory Manager

This project involved creating a simulator to translate logical addresses to physical ones using a Translation Lookaside Buffer (TLB) and page table. It handled virtual memory of 65,536 bytes, resolved page faults via demand paging, and implemented a page-replacement algorithm. The program read logical addresses from a file and output the corresponding physical addresses and byte values.

Through this project, I gained hands-on experience with system-level memory management, reinforcing concepts from operating systems. It sharpened my problem-solving skills, improved my understanding of software design, and aligned directly with my goals to master system-level programming in computer science.

Restaurant Server System (Database Application)

For this project, I designed and implemented a restaurant server system backed by a relational database. It included creating an Entity-Relationship Diagram (ERD), defining a relational schema, populating the database, and building an interactive application to manage restaurant operations. The project involved working with SQL for data management and integrating it with an application (e.g., Java or Python) using proper authentication and access control.

This assignment strengthened my skills in database design, and data modeling. It aligned with my learning goals by deepening my understanding of software design principles, improving my system-level thinking, and enhancing my ability to solve real-world problems using structured data.
